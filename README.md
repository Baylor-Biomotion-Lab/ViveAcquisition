# ViveAcquisition
Python scripts gathering controller info
# Setting Up
1. First, install triad_openvr.
2. Ensure Vive system is up and running.
3. Controllers are marked and labeled as C1 and C2. C1 is the left controller and C2 is the right controller. 
4. Test the controllers by setting `trialName = 'Test'`. Further down, plot position 1 by setting the `drawnow` command as `drawnow(position1)`.
